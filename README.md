This is the source for AnonyBits, a web app that lets you send BTC (almost) anonymously by using XMR as a privacy bridge.

It is extremely lightweight and written in pure PHP (nothing besides libcurl required). It uses REST APIs for 2 non-KYC CEX swap providers, WizardSwap and exch. You do not need to sign up for their APIs as an API key is not required. More details can be found at the website.

Contact: hi[at]anonybits.io or find me on X (@h45hb4ng)
